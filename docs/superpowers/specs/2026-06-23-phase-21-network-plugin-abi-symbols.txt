Source: `reference/open-bamboo-networking/tests/probe_plugin.cpp`.

The Phase 21 plugin must export every symbol below. Function signatures come from the matching `reference/open-bamboo-networking/src/abi_*.cpp` definitions and `reference/open-bamboo-networking/include/obn/*.hpp` callback/type declarations. The implementation plan must copy those signatures into Rust FFI declarations or an ABI probe harness; implementers must not rediscover or reduce this list.
